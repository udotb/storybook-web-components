# Image System

`jsys-image` system provides image related components that includes a responsive image, logo image and image slider.
These components are easily accessible.

### Responsive Image

```
<responsive-img src="${src}" height="${height}" width="${width}"></responsive-img>
```
### Logo Image

```
<logo-image-component filename="${filename}" height="${height}" width="${width}"></logo-image-component>
```

### Image Slider

```
<image-slider-component></image-slider-component>
```

## Features

| Attributes            | Details                                                                               |
| --------------------- | ------------------------------------------------------------------------------------- |
| ***Responsive Image***                                                                                        |
| Src/Filename          | Any src or filename can be entered to view the image.                                 |
| height                | Any height of an image can be entered in pixels or percentage.                        |
| width                 | Any width of an image can be entered in pixels or percentage.                         |
| ***Logo Image***                                                                                              |
| Src/Filename          | Any src or filename can be entered to view the image                                  |
| height                | Any height of a logo image can be entered in pixels or percentage.                    |
| width                 | Any height of a logo image can be entered in pixels or percentage.                    |

## Packages

| Package            | Description                                                                                           |
| ------------------ | -------------------------------------------------------------------- |
| Responsive Image   | It will show the responsive image that fits on every media screen    |
| Logo Image         | It will show the logo image                                          |
| Image Slider       | It will show the images that can be changed to next one via slider   |
