# Icons System

The Icon System allows you to use icons in an easy way.

### Check Icon
```
<check-font-icon width="${width}" height="${height}" color="${iconColor}"></check-font-icon>
```

### Compare Icon
```
<compare-font-icon width="${width}" height="${height}" color="${iconColor}"></compare-font-icon>
```

### Detail View Icon
```
<detailview-font-icon width="${width}" height="${height}" color="${iconColor}"></detailview-font-icon>
```

### Edit Icon
