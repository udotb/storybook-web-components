# Navigation System

`jsys-navigation` system provides anchor links with different attributes that are easily accessible.

### Anchor Link

```
<anchor-link name="${name}" href="${href}"></anchor-link>
```

## Features

| Attributes       | Details                                       |
| -----------------| --------------------------------------------- |
| ***Anchor Link***                                                |
| name             | User can set any text for the anchor link.    |
| href             | User can set any href for the anchor link.    |
| target           | User can set any target for the anchor link.  |

## Packages

| Package          | Description                                                                                           |
| ---------------- | ---------------------------------------------- |
| Anchor Link      | It will navigate to the required path or url   |
| Link Text        | It will display the clickable link text        |
