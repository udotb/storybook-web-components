# Review Stars System

`jsys-review` system provides review starts and review filter components that are easily accessible.

### Review Stars

```
<review-stars></review-stars>
```
### Review Filter

```
<review-filter-component></review-filter-component>
```


## Packages

| Package            | Description                                                                                           |
| ------------------ | ------------------------------------------------------------- |
| Review Stars       | It will show review stars                                     |
| Review Filter      | It will show the filtered results according to stars selected |
